Autenticação

# Header (recomendado)
X-API-Key: sua-chave-aqui
# Query param (alternativo)
GET /api/info?url=...&api_key=sua-chave

Endpoints

GET /api/info
Parâmetros: ?url={tiktok_url}
→ JSON: id, title, duration, uploader, thumbnail, view_count, like_count
POST /api/download
Body: {"url": "https://tiktok.com/..."}
→ Arquivo MP4 sem marca d'água (Content-Type: video/mp4)
POST /api/thumbnail
Body: {"url": "https://tiktok.com/..."}
→ Imagem JPEG do 1º frame (Content-Type: image/jpeg)

Exemplo — n8n (HTTP Request node)

# Para receber o 1º frame como imagem binária:
Method: POST
URL: http://SEU_IP:8000/api/thumbnail
Headers:
X-API-Key: sua-chave
Content-Type: application/json
Body (JSON):
{"url": "{{ $json.tiktok_url }}"}
# Response Type: selecione "File"
# O binário da imagem estará em $binary.data
